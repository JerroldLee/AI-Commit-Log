# clog-ai

## Installation

```bash
npm install -g ai-commit-log
```

## Usage

```bash
clog-ai init
```

then edit config file:

```
{
  "language": "zh or en",
  "azure_api_key": "xxxxx",
  "azure_deployment_id": "xxxx",
  "azure_base_url": "https://xxxxxxx.openai.azure.com",
  "azure_model": "gpt-3.5-turbo-16k",
  "azure_api_version": "",
}
```

## Run

```bash
clog-ai
```
